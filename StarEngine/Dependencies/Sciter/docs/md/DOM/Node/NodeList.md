# class NodeList

Represents child nodes of an element. Supports for/of enumeration of nodes.

## properties:

### length

read-only, integer, number of child nodes. 

## methods:

### item()

```js
element.childNodes.item(n:integer): Node
```

returns Nth node of the element.
